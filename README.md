## Desafio Banco - DIO

**Projeto feito com Java 21**

## Sobre o Projeto
Basicamente um projeto para aplicar conceitos de POO, simulando um banco digital.

## Tecnologias utilizadas

- Java
- Lombok

## Iniciando projeto
Para iniciar a aplicação basta executar a classe `Main.java`.
